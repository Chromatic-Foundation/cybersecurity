## 🌐 Securing Your Home Network
Your Wi-Fi network is a common entry point for hackers. Securing it is crucial to protect all connected devices.  

---

### 🔑 Router Security
- Change default admin passwords  
- Update firmware regularly  
- Disable WPS (Wi-Fi Protected Setup)  
- Use WPA3 encryption if available  

---

### 🛡 Devices & IoT
- Smart TVs, cameras, and voice assistants can be hacked if not secured  
- Change default passwords on all smart devices  
- Isolate IoT devices on a separate network if possible  

---

### 🔍 Monitor Network Activity
- Regularly check connected devices  
- Use network monitoring tools to spot unusual activity  
- Disconnect unknown devices immediately  

---

Network security protects not only your computers and phones, but all devices connected to your home network.
